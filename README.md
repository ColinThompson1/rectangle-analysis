# Analyze Rectangles

### About

Analyze rectangles to see if they:
- intersect
- contain one another
- are adjacent

### Usage

Install packages
`npm i`	

Start
`npm start src`

Test
`npm t`

Finally, just follow the prompts

### Assumptions

Rectangles are parallel with respect to x and y axis.
A contained rectangle also qualifies as intersection.
